# Task 4: Classification with Logistic Regression

## Objective
Build a binary classification model using Logistic Regression.

## Dataset
Breast Cancer Wisconsin Dataset (Scikit-Learn)

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Loaded the dataset
2. Performed train-test split
3. Standardized features
4. Trained Logistic Regression model
5. Generated predictions
6. Evaluated model using:
   - Confusion Matrix
   - Precision
   - Recall
   - F1-Score
   - ROC-AUC Score
7. Visualized ROC Curve

## Machine Learning Algorithm
Logistic Regression

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Results
Successfully classified breast cancer samples into malignant and benign categories.

## Output
- Confusion Matrix
- ROC Curve
- Classification Report
